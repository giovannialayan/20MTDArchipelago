# Stella

# How does Archipelago change this game?

Decks other than the ones you start with and all cards and elements will be locked behind checks.
Checks in Stella appear in the shops between lightyears, each deck + lightyear + difficulty combination has a ceratin number of checks.
Deck starting cards are normal.
All decks and difficulties will be playable during Archipelago whether or not you have unlocked them in your save file.
